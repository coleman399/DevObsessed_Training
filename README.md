# DevObsessed Training

Coursework for **DevObsessed's 5-week AI Development course**. Each week ships a self-contained, end-to-end project in its own folder — full backend, frontend, tests, and runbook.

## Structure

Every homework is fully self-contained: its own solution, `global.json`, `.gitignore`, `.editorconfig`, dependencies, databases, and documentation. Nothing crosses the folder boundary, so weeks can evolve independently without entangling later assignments.

```text
DevObsessed_Training/
├─ homework_one/          # Welcome App — ASP.NET Core 8 + React/Vite dark auth flow
├─ homework_two/          # (upcoming)
├─ homework_three/        # (upcoming)
├─ homework_four/         # (upcoming)
└─ homework_five/         # (upcoming)
```
